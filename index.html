Just Chat!

Just chat is a simple online chat web application made using Vue.js and Firebase.

The goal of this project was to keep the UI in sync with the Firebase database whenever a new message is added. I also used moments.js to format the time/date, and the vue-chat-scroll module.

  • Vue.js
  • JavaScript (ES6+)
  • Google Firebase
  • Materialize CSS
  • HTML
